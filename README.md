# HR-Analytics-Dashboard-
HR Analytics Dashboard Using PowerBi


OBJECTIVE
A HR Analytics Dashboard to help an organisation to improve Employee Performance
and Employee Retention (reduce attrition).

Dashboard consists of three departments
Human Resources 
Research and Development
Sales

Dashboard filters and gets customised based on the Department.

KPIs - Key Performance Indicators showing a summary of the entire dashboard.

SCOPE:
Factors leading to employee attrition
